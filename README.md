# LCP Research - Modelling Infall Time
This project models a continuous functional form for galaxy infall times defined from one virial radius of the host cluster.
By Rizkallah Botros, after first year of undergraduate study, supervised by Dr. Laura C. Parker.
All files were used in a summer research internship with the LCP Research Group at McMaster University. 
Workflow: form fit -> coefficient refit -> analysis
